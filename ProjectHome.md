# Open File Explorer #
### An _open source_ file explorer for Android ###

## Goals ##

  1. Usability
  1. Close alignment with [Android UI Design guidelines](http://developer.android.com/design/index.html)
  1. Speed


**And most importantly - NO ADs!**
